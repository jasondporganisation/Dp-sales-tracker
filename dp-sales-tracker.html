Firebase Setup Required

One-time setup to enable real-time data sync across all devices

Why Firebase? It's the free cloud database that lets all 14 agents log cases from their phones and have managers see everything live.

Step 1 — Create a Firebase project

  1. Go to console.firebase.google.com
  2. Click Add project → name it dp-sales-tracker
  3. Disable Google Analytics (not needed) → Create project

Step 2 — Enable Firestore

  1. In your project → Build → Firestore Database
  2. Click Create database → choose Start in test mode
  3. Select asia-southeast1 (Singapore) for best speed

Step 3 — Get your config

  1. In your project → Project Settings (gear icon)
  2. Scroll to Your apps → click the </> (Web) icon
  3. Register app as dp-web → copy the firebaseConfig object

Step 4 — Paste your config below

Firebase free tier supports up to 50,000 reads/day — more than enough for your agency.
